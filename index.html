SCSS Exercise: "Flow Text" Notes

This animation is accomplished by creating a @keyframs rule that the string value of selected element(s) (.character/span) gradually change from the current style to the new style at defined steps and time frame. No Javascript or Flash is used

Animation properties used:

  • animation name: "flow"
  • animation duration: 5 seconds
  • transition-timing-fucntion: ease-in-out
  • animation-iteration-count: infinite
  • animation-delay*

*Animation-delay is applied to each character (.character/span). Delay propert is set to 0.5 seconds. nth-child() selecter is used to pick these characters individually

*Note that the display value has to be "inline-block, where the top and bottoms margins/paddings are respected. Inline-block does not add a line-break after each element so each character get to be sitting next to each other"
